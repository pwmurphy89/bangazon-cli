# Bangazon


1. Clone repo
2. Install dependencies
3. RUN SCRIPT: node db/build-database.js
4. RUN SCRIPT: node bin/bangazon
5. Register as new user or login as existing user: all passwords for users are "password"
5. ctrl-C to exit program
